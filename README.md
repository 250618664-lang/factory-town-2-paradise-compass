# Factory Town 2: Paradise Compass

An unofficial, source-backed English guide hub for Factory Town 2: Paradise, covering launch status, the demo, starting modes, confirmed systems, and official updates without presenting unsupported gameplay details as fact.

## What's here

- 10 canonical pages covering release status, demo, modes, systems, and updates
- All facts are sourced from official Steam listing or official developer posts
- No game images or first-party analytics; each page contains one labeled sponsored Native Banner
- Light island-workshop visual theme

## Build

```bash
npm install
npm run build
npm run check
npm run validate
```
